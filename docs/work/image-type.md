项目中使用html2canvas将网页生成缩略图使用，当使用png格式时，基本上生成的图片都是以m为单位，不仅浪费资源，而且加载慢，影响客户体验，于是将格式改成了jpeg，相比png格式的图片，压缩了接近10倍。

## jpg和png的区别

- jpg和jpeg是一样的，是有损压缩（图片的色彩信息保留较好），体积小，网页加载速度快，但质量会不好
- png是无损压缩，体积较大，会导致网页加载慢，图片只来给你好
