жӢҶ PRODUCT.md е’Ң TECH.md#

Warp зҡ„ spec-driven workflow жҠҠи®ҫи®Ўж–ҮжЎЈжӢҶжҲҗдәҶдёӨд»ҪпјҡPRODUCT.md е’Ң TECH.mdгҖӮ

PRODUCT.md еҸӘеҶҷд»Һз”ЁжҲ·/и°ғз”Ёж–№и§’еәҰзңӢеҲ°зҡ„дёңиҘҝгҖӮз”ЁжҲ·зңӢеҲ°д»Җд№ҲгҖҒиҫ“е…Ҙд»Җд№ҲгҖҒиҫ“еҮәд»Җд№ҲгҖҒжңүе“ӘеҮ з§ҚзҠ¶жҖҒгҖҒй”ҷиҜҜзҡ„ж—¶еҖҷжҖҺд№ҲиЎЁзҺ°гҖҒиҫ№з•Ңжғ…еҶөгҖҒеҝ…йЎ»дҝқжҢҒзҡ„дёҚеҸҳйҮҸгҖӮ

е®ғзҡ„ SKILL.md йҮҢеҶҷеҫ—еҫҲжӯ»жқҝпјҡдёҚиҰҒеҶҷе®һзҺ°з»ҶиҠӮгҖҒдёҚиҰҒеҶҷеҶ…йғЁж•°жҚ®з»“жһ„гҖҒдёҚиҰҒеҶҷжЁЎеқ—иҫ№з•ҢгҖҒдёҚиҰҒеҶҷз®—жі•гҖӮиҰҒеҶҷе°ұеҶҷвҖқз”ЁжҲ·жҢүиҝҷдёӘй”®д»ҘеҗҺдјҡеҸ‘з”ҹд»Җд№ҲвҖқгҖӮ

TECH.md жүҚеҶҷе®һзҺ°пјҡеҪ“еүҚзі»з»ҹй•ҝд»Җд№Ҳж ·пјҢиҰҒж”№е“Әдәӣж–Үд»¶пјҲеёҰиЎҢеҸ·пјүпјҢж•°жҚ®жҖҺд№ҲжөҒпјҢзҠ¶жҖҒж”ҫеңЁе“Әе„ҝпјҢйЈҺйҷ©еңЁе“Әе„ҝпјҢжҖҺд№ҲйӘҢиҜҒдә§е“ҒиЎҢдёәгҖӮ

иҝҷдёӘжӢҶжі•зңӢзқҖжІЎд»Җд№ҲпјҢдҪҶе®ғи§ЈеҶідәҶдёҖдёӘжҲ‘д»¬еҶҷи®ҫи®Ўж–ҮжЎЈзҡ„иҖҒжҜӣз—…вҖ”вҖ”дә§е“ҒиЎҢдёәе’Ңе®һзҺ°з»ҶиҠӮж··еңЁдёҖиө·гҖӮ

ж··еңЁдёҖиө·зҡ„и®ҫи®Ўж–ҮжЎЈеҫҲйҡҫзңӢгҖӮдә§е“ҒеҗҢеӯҰзңӢдёҚжҮӮпјҢеӣ дёәйҮҢйқўдёҖеҚҠжҳҜ traitгҖҒArcгҖҒasync иҝҷз§ҚдёңиҘҝгҖӮе·ҘзЁӢеёҲеҸҲи§үеҫ—йӮЈдәӣиЎҢдёәжҸҸиҝ°еӨӘиҷҡпјҢзңӢе®ҢдёҚзҹҘйҒ“еҲ°еә•иҜҘжҖҺд№ҲеҶҷгҖӮжңҖеҗҺж•ҙд»Ҫж–ҮжЎЈеҸҳжҲҗдёҖдёӘд»ӘејҸпјҡеҶҷзҡ„ж—¶еҖҷз—ӣиӢҰпјҢеҶҷе®ҢжІЎдәәзңӢпјҢд»Јз Ғж”№дәҶд№ӢеҗҺд№ҹжІЎдәәеӣһеҺ»жӣҙж–°гҖӮ

Warp иҝҷиҫ№жҠҠе®ғд»¬еҲҶжҲҗдёӨдёӘж–Үд»¶д№ӢеҗҺпјҢдә§е“ҒиЎҢдёәеҸҜд»ҘдёҖдёӘдёҖдёӘеҲ—жҲҗ numbered invariantпјҢзұ»дјјпјҡ

жҲ‘зү№еҲ«е–ңж¬ўиҝҷдёӘи®ҫи®Ўзҡ„дёҖдёӘз»ҶиҠӮпјҡPRODUCT.md йҮҢ SKILL.md жҳҺзЎ®зҰҒжӯўеҶҷ Validation з« иҠӮпјҢйӘҢиҜҒе…ЁйғЁеҪ’еҲ° TECH.mdгҖӮиҝҷж ·е°ұдёҚдјҡеҮәзҺ°вҖқдә§е“Ғж–ҮжЎЈйҮҢеҶҷдәҶдёҖе ҶжөӢиҜ•з”ЁдҫӢпјҢдҪҶе’Ң TECH.md йҮҚеӨҚвҖқиҝҷз§ҚдәӢгҖӮ

еҸҰеӨ– SKILL.md йҮҢеҶҷдәҶдёҖеҸҘжҲ‘и§үеҫ—еҫҲе…ій”®зҡ„иҜқпјҡе®һзҺ°иҝҮзЁӢдёӯеҰӮжһңеҒҸзҰ»дәҶ specпјҢиҰҒеӣһеҺ»ж”№ specпјҢи®© spec е’ҢжңҖз»Ҳ ship зҡ„д»Јз ҒдҝқжҢҒдёҖиҮҙпјҢspec е’Ңе®һзҺ°ж”ҫеңЁеҗҢдёҖдёӘ PR йҮҢ mergeгҖӮ

иҝҷзӣёеҪ“дәҺжҠҠи®ҫи®Ўж–ҮжЎЈеҸҳжҲҗ living documentationгҖӮж–ҮжЎЈдёҚжҳҜе®Ўжү№жқҗж–ҷпјҢжҳҜе®һзҺ°иҝҮзЁӢзҡ„дёҖйғЁеҲҶгҖӮд»Јз Ғж”№дәҶпјҢж–ҮжЎЈи·ҹзқҖж”№пјҢPR дёҖиө·еҗҲгҖӮиҝҷжҳҜеӨ§йғЁеҲҶеӣўйҳҹеҒҡдёҚеҲ°гҖҒдҪҶе…¶е®һжІЎд»Җд№ҲжҠҖжңҜй—Ёж§ӣзҡ„дәӢгҖӮ

жҖ»жҺ§ + йҳ¶ж®ө#

skill еӨҡдәҶд№ӢеҗҺпјҢWarp з”ЁдәҶдёҖдёӘжҲ‘и§үеҫ—жҢәеҖјеҫ—жҠ„зҡ„з»“жһ„пјҡжҖ»жҺ§ skill еҠ йҳ¶ж®ө skillгҖӮ

жҖ»жҺ§ skill жҳҜ spec-driven-implementationгҖӮе®ғиҮӘе·ұдёҚеҶҷд»Јз ҒгҖҒдёҚејҖ PRгҖҒдёҚи·‘жөӢиҜ•гҖӮе®ғеҸӘеҒҡеҮ д»¶дәӢпјҡ

  • еҲӨж–ӯиҝҷдёӘд»»еҠЎеҲ°еә•йңҖдёҚйңҖиҰҒеҶҷ specпјӣ
  • еҰӮжһңйңҖиҰҒпјҢе…ҲеҶҷ PRODUCT иҝҳжҳҜе…ҲеҶҷ TECHпјӣ
  • еҶҷе®Ң spec д№ӢеҗҺи·іеҲ° implement-specsпјӣ
  • е®һзҺ°иҝҮзЁӢдёӯеҰӮжһңеҒҸдәҶпјҢеӣһеӨҙж”№ specпјӣ
  • жңҖеҗҺи·іеҲ° create-prгҖӮ

йҳ¶ж®ө skill еҗ„иҮӘиҙҹиҙЈиҮӘе·ұйӮЈж®өгҖӮwrite-product-spec еҸӘз®ЎеҶҷ PRODUCT.mdпјҢwrite-tech-spec еҸӘз®ЎеҶҷ TECH.mdпјҢcreate-pr еҸӘз®ЎејҖ PR еүҚзҡ„йӮЈдёҖеқЁжЈҖжҹҘгҖӮ

жҲ‘иҮӘе·ұд№ӢеүҚзңӢиҝҮдёҖдәӣеӣўйҳҹе°қиҜ•жҗһ prompt е·ҘзЁӢпјҢеҒҡжі•жҳҜеҶҷдёҖдёӘи¶…еӨ§зҡ„ system promptпјҢжҠҠжүҖжңүи§„еҲҷйғҪеЎһиҝӣеҺ»пјҢжҒЁдёҚеҫ—дёҖдёӘж–Үд»¶з®Ўе®ҢжүҖжңүдәӢгҖӮз»“жһңе°ұжҳҜ prompt и¶ҠеҶҷи¶Ҡй•ҝпјҢи§„еҲҷд№Ӣй—ҙдә’зӣёжү“жһ¶пјҢagent иҜ»е®Ңд№ӢеҗҺдёҚзҹҘйҒ“еҪ“еүҚеҲ°еә•иҜҘжҢүе“ӘжқЎеҒҡгҖӮ

жҠҠжөҒзЁӢжӢҶжҲҗжҖ»жҺ§ + йҳ¶ж®өпјҢжҜҸдёӘж–Үд»¶еҸӘеӣһзӯ”дёҖдёӘиҢғеӣҙзҡ„й—®йўҳпјҢжҳҜи§„жЁЎеҢ–зҡ„еүҚжҸҗгҖӮиҝҷе…¶е®һе’ҢжҲ‘д»¬еҶҷд»Јз Ғж—¶зҡ„жЁЎеқ—жӢҶеҲҶжҳҜдёҖеӣһдәӢпјҢеҸӘдёҚиҝҮзҺ°еңЁиҝҷдәӣжЁЎеқ—зҡ„вҖқи°ғз”Ёж–№вҖқжҳҜ agentгҖӮ

жҠҠ spec-driven-implementation йҮҢзҡ„еҲӨж–ӯз”»еҮәжқҘеӨ§жҰӮжҳҜиҝҷж ·пјҡ

жҜҸдёӘеҲҶж”ҜйғҪеҜ№еә”дёҖд»ҪеҫҲе…·дҪ“зҡ„еҲӨж–ӯж ҮеҮҶвҖ”вҖ”вҖң1k+ LOCвҖқгҖҒвҖңи·ЁжЁЎеқ—вҖқгҖҒвҖңиЎҢдёәжңүжӯ§д№үвҖқпјҢдёҚжҳҜжЁЎзіҠзҡ„вҖқйҮҚиҰҒзҡ„еҠҹиғҪвҖқгҖӮиҝҷз§ҚдёңиҘҝеҰӮжһңеҸӘжҳҜеҶҷеңЁ wiki йҮҢпјҢи°Ғд№ҹдёҚдјҡжҢүе®ғжқҘеҲӨж–ӯпјӣеҶҷеңЁ SKILL.md йҮҢиў« agent еҠ иҪҪпјҢеҸҚиҖҢжҜҸдёҖж¬ЎйғҪеҫ—иө°дёҖйҒҚгҖӮ

йҖҡз”Ёи§„еҲҷе’Ңжң¬ең°и§„еҲҷ#

Warp зҡ„ skills йҮҢиҝҳжңүдёҖеҜ№жҲ‘и§үеҫ—еҫҲеҖјеҫ—еҖҹйүҙзҡ„и®ҫи®Ўпјҡ

  • review-pr
  • review-pr-local

review-pr еҶҷйҖҡз”Ёзҡ„ review жөҒзЁӢпјҡжҖҺд№ҲиҜ» diffгҖҒиҫ“еҮәж јејҸжҳҜд»Җд№ҲгҖҒseverity жҖҺд№ҲеҲҶгҖӮ

review-pr-local еҸӘеҶҷеҪ“еүҚиҝҷдёӘ repo иҮӘе·ұзҡ„жҖӘи§„зҹ©пјҡWarp е®ўжҲ·з«ҜжҳҜ Rust еҠ иҮӘз ” UI жЎҶжһ¶пјҢжңүдәӣеҶҷжі•еңЁеҲ«еӨ„еҗҲзҗҶдҪҶеңЁиҝҷдёӘ repo йҮҢзҰҒз”Ёпјӣжҹҗдәӣзӣ®еҪ•жңү ownerпјҢPR ж”№еҲ°йӮЈйҮҢиҰҒе…Ҳ ping и°Ғпјӣжҹҗдәӣ panic еңЁ prod жҳҜдёҚиғҪе®№еҝҚзҡ„зӯүзӯүгҖӮ

жҜҸдёӘеӣўйҳҹйғҪжңүиҝҷз§ҚвҖқзҘ–дј д№ дҝ—вҖқгҖӮжңүдәӣжҳҜеҺҶеҸІеҢ…иўұз•ҷдёӢзҡ„пјҢжңүдәӣжҳҜжҹҗдёӘиҖҒе‘ҳе·ҘжҹҗеӨ©дёӢеҚҲйҡҸжүӢе®ҡзҡ„еҒҸеҘҪгҖӮж— и®әеҗҲзҗҶдёҚеҗҲзҗҶпјҢеҸӘиҰҒе®ғзңҹе®һеҪұе“Қ reviewпјҢе°ұеә”иҜҘиў«еҶҷеҮәжқҘгҖӮ

дҪҶдёҚиҰҒеЎһиҝӣйҖҡз”Ё skill йҮҢгҖӮйҖҡз”Ёзҡ„йғЁеҲҶиҰҒдҝқжҢҒе№ІеҮҖпјҢжң¬ең°е·®ејӮеҸҰиө·дёҖдёӘж–Үд»¶гҖӮжҲ‘зңӢиҝҮеӨӘеӨҡеӣўйҳҹжңҖеҗҺеҶҷеҮәжқҘзҡ„вҖқејҖеҸ‘и§„иҢғвҖқжҳҜдёҖдёӘ 5000 еӯ—зҡ„ markdownпјҢеүҚ 1000 еӯ—жҳҜйҖҡз”ЁеҒҡжі•пјҢеҗҺ 4000 еӯ—жҳҜеҗ„з§Қзү№дҫӢе’ҢвҖқеҰӮжһңдҪ еңЁ X жңҚеҠЎйҮҢпјҢиҜ·ж”№жҲҗ YвҖқгҖӮиҝҷз§Қж–ҮжЎЈжІЎдәәзңӢеҫ—еҠЁгҖӮ

жӣҙеҗҲзҗҶзҡ„еҒҡжі•жҳҜеҲҶеұӮпјҢжҜ”еҰӮпјҡ

йҖҡз”Ёзҡ„йғЁеҲҶеҸҜд»ҘеңЁеӨҡдёӘд»“еә“д№Ӣй—ҙеӨҚз”ЁпјҢжң¬ең°е·®ејӮеҗ„иҮӘз»ҙжҠӨпјҢдә’дёҚжұЎжҹ“гҖӮ

е·ҘзЁӢйҮҢеҫҲеӨҡжҠҪиұЎеҒҡеқҸдәҶпјҢжҳҜеӣ дёәдёҖејҖе§ӢеҒҮиЈ…еӨ§е®¶йғҪдёҖж ·гҖӮжңҖеҗҺйҖҡз”ЁжөҒзЁӢиў«еҗ„з§Қзү№дҫӢеЎһзҲҶпјҢжІЎжі•зңӢпјҢеӣўйҳҹеҸҲйҖҖеӣһеҲ°еҸЈеҸЈзӣёдј гҖӮ

д»Җд№Ҳж—¶еҖҷдёҠи„ҡжң¬#

Warp зҡ„ skills еӨ§йғЁеҲҶйғҪжҳҜзәҜ markdownгҖӮresolve-merge-conflicts жҳҜе°‘ж•°еёҰи„ҡжң¬зҡ„ skill д№ӢдёҖгҖӮеҺҹеӣ д№ҹеҫҲз®ҖеҚ•вҖ”вҖ”йқ  agent жүӢе·ҘиҜ»дёҖдёӘеҮ еҚғиЎҢзҡ„еҶІзӘҒж–Үд»¶еҺ»жҠҪ ours/base/theirsпјҢиҰҒд№ҲзҲҶдёҠдёӢж–ҮпјҢиҰҒд№ҲжјҸ hunkгҖӮи„ҡжң¬дёҖи·‘е°ұиғҪзЁіе®ҡиҫ“еҮәеҮ дёӘзҙ§еҮ‘зҡ„ diffпјҢagent жӢҝзқҖиҝҷдёӘ diff еҺ»жғіжҖҺд№Ҳ merge е°ұиЎҢгҖӮ

е®ғзҡ„ж•ҙдҪ“жөҒзЁӢй•ҝиҝҷж ·пјҡ

жіЁж„Ҹи„ҡжң¬еҸӘиҙҹиҙЈвҖқзЁіе®ҡжҠҪеҮәеҶІзӘҒдёҠдёӢж–ҮвҖқиҝҷдёҖд»¶дәӢпјҢmerge еҶізӯ–жң¬иә«иҝҳжҳҜ agent еңЁеҒҡгҖӮиҝҷдёӘиҫ№з•ҢеҲ’еҫ—еҫҲйҮҚиҰҒпјҡи„ҡжң¬еҸӘиЎҘвҖқagent еҚ•и·‘жҗһдёҚе®ҡзҡ„йӮЈйғЁеҲҶвҖқпјҢеү©дёӢзҡ„еҲӨж–ӯд»Қз„¶еңЁ markdown йҮҢжҸҸиҝ°гҖӮ

жҲ‘жҠҠиҝҷдёӘеҲӨж–ӯжҖ»з»“дёҖдёӢпјҢеӨ§жҰӮжҳҜиҝҷж ·пјҡ

  • еҸӘжҳҜжөҒзЁӢзәҰе®ҡпјҢдёҚйңҖиҰҒзІҫзЎ®и§Јжһҗд»Җд№ҲдёңиҘҝпјҢmarkdown е°ұеӨҹдәҶпјӣ
  • йңҖиҰҒйҮҚеӨҚи·‘е‘Ҫд»ӨпјҢжҠҠе‘Ҫд»ӨзӣҙжҺҘеҶҷеңЁ markdown йҮҢпјҢи®© agent иҮӘе·ұеӨҚеҲ¶еҺ»и·‘пјӣ
  • йңҖиҰҒзЁіе®ҡең°д»ҺеӨ§ж–Үд»¶гҖҒеӨ§иҫ“еҮәйҮҢжҠҪз»“жһ„еҢ–дҝЎжҒҜпјҢеҶҷи„ҡжң¬пјҢи®© skill и°ғи„ҡжң¬пјӣ
  • йңҖиҰҒй•ҝж—¶й—ҙиҝҗиЎҢгҖҒи°ғеәҰгҖҒйҡ”зҰ»зҺҜеўғгҖҒж”¶йӣҶдә§зү©пјҢеҶҚиҖғиҷ‘ harness йӮЈдёҖеҘ—гҖӮ

дёҚиҰҒдёҖдёҠжқҘе°ұ harnessгҖӮ

жҲ‘д№ӢеүҚеҶҷиҝҮдёҖзҜҮдё“й—ЁиҒҠ harnessпјҲharness-or-not-harnessпјүпјҢи§ӮзӮ№д№ҹжҳҜзұ»дјјзҡ„пјҡharness и§ЈеҶізҡ„жҳҜвҖқagent еҚ•и·‘жҗһдёҚе®ҡзҡ„е·ҘзЁӢй—®йўҳвҖқпјҢдёҚжҳҜвҖқжҲ‘д»¬зҡ„жөҒзЁӢжІЎеҶҷжҳҺзҷҪвҖқгҖӮеңЁжөҒзЁӢжң¬иә«йғҪжІЎжІүж·Җзҡ„ж—¶еҖҷеҺ»жҗһ harnessпјҢжң¬иҙЁдёҠжҳҜеңЁз»ҷж··д№ұеҘ—дёӘеЈіпјҢи·‘иө·жқҘеҸҚиҖҢжӣҙйҡҫи°ғгҖӮ

е…ҲжҠҠжөҒзЁӢеҶҷжҳҺзҷҪпјҢеҶҚзңӢе“ӘдәӣзҺҜиҠӮзңҹзҡ„йңҖиҰҒе·Ҙе…·ж”Ҝж’‘гҖӮйЎәеәҸеҸҚдәҶе°ұеҫҲйҡҫиЎҘгҖӮ

жҠ„дёҖдёӘжңҖе°ҸеҸҜз”Ёзҡ„зүҲжң¬#

еҰӮжһңдҪ жғіз»ҷиҮӘе·ұзҡ„еӣўйҳҹжҗӯдёҖеҘ—пјҢеҸҜд»Ҙд»ҺиҝҷеҘ—жңҖе°ҸйӣҶеҗҲејҖе§ӢгҖӮ

жҜҸдёӘ SKILL.md иҮіе°‘иҰҒеӣһзӯ”дёӢйқўеҮ дёӘй—®йўҳпјҢеҶҷдёҚеҮәжқҘе°ұиҜҙжҳҺиҝҷдёӘ skill иҝҳжІЎжғіжё…жҘҡпјҡ

  • д»Җд№Ҳжғ…еҶөдёӢз”ЁпјҹпјҲеҶҷеңЁ description йҮҢпјү
  • иҫ“е…ҘжҳҜд»Җд№ҲпјҹпјҲз”ЁжҲ·зҡ„иҜ·жұӮ + е“Әдәӣе·Іжңүж–Үд»¶ + е“Әдәӣд»Јз ҒиҰҒе…ҲиҜ»пјү
  • иҫ“еҮәжҳҜд»Җд№ҲпјҹпјҲз”ҹжҲҗжҲ–дҝ®ж”№дәҶе“Әдәӣж–Үд»¶пјү
  • еҝ…йЎ»еҒҡе“ӘдәӣжӯҘйӘӨпјҹпјҲжҢүйЎәеәҸеҶҷпјү
  • еӨұиҙҘд№ӢеҗҺи·іеҲ°е“ӘдёӘ skillпјҹ

жңҖеҗҺдёҖжқЎе°Өе…¶е®№жҳ“иў«еҝҪз•ҘгҖӮдёҖдёӘ skill еҶҷе®Ңд№ӢеҗҺеӨ§жҰӮзҺҮдёҚжҳҜз»ҲзӮ№пјҢиҰҒд№ҲиҝӣдёӢдёҖдёӘйҳ¶ж®өпјҢиҰҒд№ҲйҒҮеҲ°й”ҷиҜҜеӣһеҲ°иҜҠж–ӯжөҒзЁӢгҖӮжҠҠиҝҷдёӘи·іиҪ¬е…ізі»еҶҷжё…жҘҡпјҢж•ҙеҘ— skill жүҚиғҪдёІжҲҗ workflowпјҢиҖҢдёҚжҳҜдёҖе ҶеӯӨз«Ӣзҡ„иҜҙжҳҺд№ҰгҖӮ

йҷ„пјҡеҮ дёӘ skill зҡ„еҶізӯ–еӣҫ#

дёӢйқўиҝҷеҮ еј жҳҜжҲ‘жҠҠ Warp зҺ°жңү skill йҮҢвҖқеҲӨж–ӯеҲҶж”ҜжҜ”иҫғеӨҡвҖқзҡ„йӮЈдәӣжҠҪеҮәжқҘз”»зҡ„гҖӮзңӢеӣҫжҜ”зңӢ SKILL.md е…Ёж–Үжӣҙеҝ«пјҢд№ҹиғҪзңӢеҮәжҜҸдёӘ skill йҮҢвҖқд»Җд№ҲжҳҜеҝ…йЎ»еҒҡзҡ„гҖҒд»Җд№ҲжҳҜжңүжқЎд»¶еҒҡзҡ„гҖҒеӨұиҙҘд№ӢеҗҺи·іеҲ°е“ӘвҖқгҖӮ

description еӯ—ж®өзҡ„и®ҫи®Ўйқһеёёе·§еҰҷвҖ”вҖ”е®ғдёҚжҳҜз»ҷдәәиҜ»зҡ„вҖқж ҮйўҳвҖқпјҢиҖҢжҳҜз»ҷ agent иҜ»зҡ„вҖқи§ҰеҸ‘жқЎд»¶вҖқгҖӮиҝҷз§Қи®ҫи®Ўи®© agent иғҪеңЁеӨ§йҮҸ skills дёӯеҝ«йҖҹжүҫеҲ°еҪ“еүҚйңҖиҰҒзҡ„йӮЈдёҖдёӘпјҢеҗҢж—¶д№ҹејәиҝ«еҶҷ skill зҡ„дәәжҖқиҖғвҖқд»Җд№Ҳж—¶еҖҷз”ЁвҖқиҝҷдёӘй—®йўҳпјҢиҖҢдёҚжҳҜеҸӘеҶҷвҖқжҖҺд№ҲеҒҡвҖқгҖӮ

PRODUCT.md е’Ң TECH.md зҡ„еҲҶзҰ»пјҢжң¬иҙЁдёҠжҳҜе…іжіЁзӮ№еҲҶзҰ»зҡ„дҪ“зҺ°гҖӮдә§е“Ғж–ҮжЎЈеҸӘе…іеҝғвҖқз”ЁжҲ·зңӢеҲ°д»Җд№ҲвҖқпјҢжҠҖжңҜж–ҮжЎЈеҸӘе…іеҝғвҖқжҖҺд№Ҳе®һзҺ°вҖқгҖӮиҝҷз§ҚеҲҶзҰ»и®©дёӨзұ»иҜ»иҖ…пјҲдә§е“Ғз»ҸзҗҶе’Ңе·ҘзЁӢеёҲпјүйғҪиғҪзңӢжҮӮиҮӘе·ұйңҖиҰҒзҡ„йғЁеҲҶпјҢйҒҝе…ҚдәҶж–ҮжЎЈеҸҳжҲҗвҖқеҸӘжңүеҶҷзҡ„дәәз—ӣиӢҰпјҢиҜ»зҡ„дәәдёҚжҮӮвҖқзҡ„д»ӘејҸгҖӮ

жҖ»з»“#

Warp зҡ„ skill еҢ–жөҒзЁӢдҪ“зі»жҸҗдҫӣдәҶдёҖдёӘеҸҜеӨҚеҲ¶зҡ„жЁЎжқҝпјҡ

  1. з®ҖеҚ•жҖ§дјҳе…Ҳ: зәҜ markdown дёәдё»пјҢи„ҡжң¬дёәиҫ…
  2. жҳҺзЎ®зҡ„и§ҰеҸ‘жқЎд»¶: description еӯ—ж®өеҗҢж—¶иҜҙжҳҺвҖқеҒҡд»Җд№ҲвҖқе’ҢвҖқд»Җд№Ҳж—¶еҖҷз”ЁвҖқ
  3. ж–ҮжЎЈеҲҶзҰ»: PRODUCT.md дёҺ TECH.md еҗ„еҸёе…¶иҒҢ
  4. еҲҶеұӮжһ¶жһ„: жҖ»жҺ§ + йҳ¶ж®ө skill
  5. йҖҡз”ЁдёҺжң¬ең°еҲҶзҰ»: йҒҝе…Қзү№дҫӢжұЎжҹ“йҖҡз”ЁжөҒзЁӢ
  6. living documentation: ж–ҮжЎЈдёҺд»Јз ҒеҗҢ PR еҗҲе№¶

иҝҷеҘ—дҪ“зі»зҡ„ж ёеҝғд»·еҖјеңЁдәҺпјҡжҠҠеӣўйҳҹзҡ„йҡҗжҖ§зҹҘиҜҶпјҲиҖҒе‘ҳе·Ҙи„‘еӯҗйҮҢзҡ„д№ дҝ—пјүеҸҳжҲҗжҳҫжҖ§зҹҘиҜҶпјҲеҸҜжү§иЎҢзҡ„ skillпјүпјҢи®©ж–°дәәгҖҒж–°еҗҢдәӢгҖҒAI agent йғҪиғҪеҝ«йҖҹдёҠжүӢгҖӮ

жҲ‘зҡ„жү©еұ•жҖ»з»“#

и®ҫи®ЎжҖқи·Ҝзҡ„ж·ұеәҰи§ЈиҜ»#

Warp иҝҷеҘ— skill дҪ“зі»зҡ„и®ҫи®ЎжҖқи·ҜеҸҜд»Ҙд»ҺеҮ дёӘеұӮйқўдҪ“зҺ°дәҶе·ҘзЁӢжҷәж…§пјҡ

  1. **д»ҺвҖқз»ҷдәәеҶҷзҡ„ж–ҮжЎЈвҖқеҲ°вҖқз»ҷ agent зҡ„еҸҜжү§иЎҢж–Үд»¶вҖқзҡ„иҪ¬еҸҳ

    • дј з»ҹж–ҮжЎЈпјҡжҸҸиҝ°жҖ§зҡ„гҖҒжЁЎзіҠзҡ„гҖҒйңҖиҰҒдәәеҲӨж–ӯ
    • SkillпјҡжҢҮд»ӨжҖ§зҡ„гҖҒзІҫзЎ®зҡ„гҖҒжңүжҳҺзЎ®иҫ“е…Ҙиҫ“еҮә
    • дҪҶ skill д»Қз„¶жҳҜдәәиғҪзңӢжҮӮзҡ„ markdownпјҢдёҚжҳҜй»‘зӣ’

  2. жёҗиҝӣејҸе·Ҙе…·еҢ–

    • е…ҲжңүжөҒзЁӢжІүж·ҖпјҲmarkdownпјү
    • еҶҚзңӢе“ӘдәӣзҺҜиҠӮзңҹзҡ„йңҖиҰҒи„ҡжң¬
    • жңҖеҗҺжүҚиҖғиҷ‘ harness
    • иҝҷдёӘйЎәеәҸдёҚиғҪеҸҚиҝҮжқҘ

  3. е…іжіЁзӮ№еҲҶзҰ»

    • жҖ»жҺ§дёҺйҳ¶ж®өеҲҶзҰ»
    • йҖҡз”ЁдёҺжң¬ең°еҲҶзҰ»
    • дә§е“ҒдёҺжҠҖжңҜеҲҶзҰ»
    • иҝҷз§ҚеҲҶзҰ»и®©жҜҸдёӘйғЁеҲҶйғҪз®ҖеҚ•еҲ°еҸҜз»ҙжҠӨ

дёҺе…¶д»–жҰӮеҝөзҡ„иҒ”зі»#

иҝҷеҘ— skill дҪ“зі»и®©жҲ‘жғіеҲ°еҮ дёӘзӣёе…іжҰӮеҝөпјҡ

  1. дёҺвҖқеҹәзЎҖи®ҫж–ҪеҚід»Јз ҒвҖқзҡ„иҒ”зі»

    • Skill еҸҜд»ҘзңӢдҪңжҳҜвҖқжөҒзЁӢеҚід»Јз ҒвҖқ
    • йғҪжҳҜжҠҠд»ҘеүҚдәәе·Ҙж“ҚдҪңзҡ„дёңиҘҝеҸҳжҲҗеҸҜзүҲжң¬жҺ§еҲ¶гҖҒеҸҜйҮҚеӨҚжү§иЎҢзҡ„ж–Үд»¶

  2. дёҺ prompt engineering зҡ„еҢәеҲ«

    • еҫҲеӨҡеӣўйҳҹжҗһ prompt engineering жҳҜеҶҷдёҖдёӘи¶…еӨ§зҡ„ system prompt
    • Warp жҳҜжӢҶжҲҗе°Ҹзҡ„гҖҒжңүжҳҺзЎ®и§ҰеҸ‘жқЎд»¶зҡ„ skill
    • иҝҷе’Ңд»Јз Ғзҡ„жЁЎеқ—еҢ–жҳҜдёҖдёӘйҒ“зҗҶ

  3. дёҺ SOPпјҲж ҮеҮҶдҪңдёҡзЁӢеәҸпјүзҡ„иҒ”зі»

    • Skill жҳҜеёҰи§ҰеҸ‘жқЎд»¶зҡ„ SOP
    • дёҚд»…еҶҷвҖқжҖҺд№ҲеҒҡвҖқпјҢиҝҳеҶҷвҖқд»Җд№Ҳж—¶еҖҷеҒҡвҖқ
    • иҖҢдё”еҸҜд»Ҙиў« agent иҮӘеҠЁеҠ иҪҪжү§иЎҢ

еұҖйҷҗжҖ§зҡ„жҖқиҖғ#

иҝҷеҘ—дҪ“зі»д№ҹжңүе®ғзҡ„йҖӮз”Ёиҫ№з•Ңпјҡ

  1. йҖӮеҗҲе·Із»ҸжңүзӣёеҜ№зЁіе®ҡжөҒзЁӢзҡ„еӣўйҳҹ

    • еҰӮжһңеӣўйҳҹиҝҳеңЁжҺўзҙўйҳ¶ж®өпјҢжөҒзЁӢжң¬иә«е°ұеңЁеҸҳпјҢskill еҢ–еҸҜиғҪеҸҚиҖҢжқҹзјҡжүӢи„ҡ
    • е…ҲжңүжөҒзЁӢжІүж·ҖпјҢеҶҚ skill еҢ–

  2. йҖӮеҗҲжңүдёҖе®ҡи§„жЁЎзҡ„еӣўйҳҹ

    • е°ҸеӣўйҳҹеҸҜиғҪдёҚйңҖиҰҒиҝҷд№ҲйҮҚзҡ„дҪ“зі»
    • дҪҶеҸҜд»Ҙе…Ҳд»ҺжңҖе°ҸйӣҶеҗҲејҖе§ӢпјҢйҡҸзқҖеӣўйҳҹжҲҗй•ҝйҖҗжӯҘеўһеҠ 

  3. йңҖиҰҒеӣўйҳҹжңүеҶҷ skill зҡ„иғҪеҠӣе’Ңд№ жғҜ

    • дёҚжҳҜжүҖжңүе·ҘзЁӢеёҲйғҪиғҪжҠҠжөҒзЁӢеҶҷеҫ—жё…жҷ°гҖҒеҸҜжү§иЎҢ
    • йңҖиҰҒжңүдәәдё“й—Ёз»ҙжҠӨиҝҷеҘ—дҪ“зі»
